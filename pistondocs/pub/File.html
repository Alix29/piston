Logan.File

The File object represents a File.
It allows you create, read, write, delete or otherwise modify a file.

Constructors

Logan.File(Object propertyMap?)
Creates a new File object
Arguments
  • Object propertyMap (optional)
    The initial set of Properties to set
Example
var f = new Logan.File();
f.path = "text.txt";
Example
var f = new Logan.File({ "path" : "text.txt" });
Logan.File(String path, Object propertyMap?)
Creates a new File object with the given path
Arguments
  • String path
    The full or relative path to the file
  • Object propertyMap (optional)
    The initial set of Properties to set
Example
var f = new Logan.File("test.txt");

Properties

String path The full path to the file
Valid Values
  • null (default)
  • *
read+write
