Logan.Directory

The Directory object represents a Directory.
It allows you list sub directorys and directories.

Constructors

Logan.Directory(Object propertyMap?)
Creates a new Directory object
Arguments
  • Object propertyMap (optional)
    The initial set of Properties to set
Example
var f = new Logan.Directory();
f.path = "/tmp";
Example
var f = new Logan.Directory({ "path" : "/tmp" });
Logan.Directory(String path, Object propertyMap?)
Creates a new Directory object with the given path
Arguments
  • String path
    The full or relative path to the directory
  • Object propertyMap (optional)
    The initial set of Properties to set
Example
var f = new Logan.Directory("/tmp");

Properties

String path The full path to the directory
Valid Values
  • null (default)
  • *
read+write
